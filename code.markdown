### __A small python example__
A simple __python__ code block, showing a typical python pattern.
(Code to be replaced with actal code!)

```python
#!/usr/bin/env python3
import sys

def hello():
    greetings = ['Hello', 'Hola', 'Bonjour', 'Hallo','Ciao', 'Ni Hao']

    for greeting in greetings:
        print(f'{greeting}, World!')


def main():
    hello()
    return 0


if __name__ == '__main__':
    sys.exit(main())
```
